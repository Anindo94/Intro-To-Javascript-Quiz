Quiz-1
Declare a function called laugh() that returns "hahahahahahahahahaha!".
 Print the value returned from the laugh() function to the console.

 Code:
 function laugh() {
return "hahahahahahahahahaha!";
}

console.log(laugh());

Output:
hahahahahahahahahaha!
